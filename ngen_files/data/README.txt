For use in NextGen, the lstm_py project folder should be copied into
the NextGen project folder, at ngen/extern/lstm_py, and the files in
this "ngen_files" folder should be copied into the ngen project folder
in the indicated subfolders: e.g. ngen/data/forcings or ngen/data/lstm.
